# Bazar
Представляет собой веб-сайт доску для размещений объявлений с возможностью отклика на них и фильтрацией. Авторизация в данном пространстве происходит через почту/логин вуза, чтобы обеспечить безопасность и закрытость системы. Целевая аудитория – студенты, живущие в общежитие.

## Цель проекта
Разработать прототип доски-объявлений, который позволяет студентам, проживающим в общежитиях, решать проблему перегруженности внутренних чатов и рынка сбыта б/у товаров.
## Технологии
- Frontend: HTML, CSS, JavaScript
- Backend: Python
- Базы данных: SQLite
- Инструменты: Figma, GitHub, Adobe Photoshop
